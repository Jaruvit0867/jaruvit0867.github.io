Gesture Presenter

A browser-based presentation controller that lets users navigate PDF slides with webcam hand gestures. It combines MediaPipe hand tracking, PDF rendering, and mobile remote control into a polished presentation workflow.

  • Gesture-driven slide navigation with real-time feedback
  • Browser-first privacy model with no document upload required
  • Remote control support through QR-based phone pairing

Gorilla AI Assistant

A full-stack chat application that pairs a Spring Boot backend with a Next.js frontend and Azure AI Foundry integration. The project focuses on admin authentication, session-driven workflows, and a clear split between frontend and backend responsibilities.

  • Spring Boot 4 API with secured session-based authentication
  • Next.js 16 interface for chat and admin workflows
  • Azure AI Foundry integration for assistant capabilities

EatEasy Food Order

A Thai restaurant ordering system with a Next.js frontend, FastAPI backend, analytics, kitchen operations, and a local print app for in-store execution. It is built around a practical single-restaurant deployment model.

  • Thai voice ordering and menu matching workflow
  • Kitchen dashboard, admin tools, and order analytics
  • Hybrid deployment model with cloud backend and local print app
